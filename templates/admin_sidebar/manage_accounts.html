Manage Accounts

This section will allow you to manage both resident and admin accounts.

Resident Accounts Management

  • View all residents
  • Search by unit/building
  • Add/edit resident info
  • Deactivate resident account (optional soft-delete)

Admin Accounts Management

  • View list of all admins
  • Create new admin account (username, email, password, name, created_by = current user_id)
  • Deactivate admin (soft delete or restrict access)
  • View who created which admin (use created_by)
